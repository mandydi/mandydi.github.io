博客搭建详情

  • 自主搭建hugo

    1. 安装hugo

    2. 用hugo生成博客

    3. 下载并配置主题

    4. 在本地启动个人博客

      博客根目录下克隆

      hugo server -t hugo-theme-ztyblog --buildDrafts

    5. 实际来写一遍

    6. 将个人博客部署到远端服务器

      hugo --theme=m10c --baseUrl="https://unniWang/gitstudy/" --buildDrafts

      会生成一个public文件

      进入Public

      git init

      git add .

      git commit -m ""

      git remote –theme=m10c -baseUrl="" –buildDrafts

      git push -u origin master

    7. 本地仓库部署到博客(Gitee)
